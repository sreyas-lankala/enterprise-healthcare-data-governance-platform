# Enterprise Healthcare Data Governance Platform

This project simulates a real-world enterprise healthcare data platform focused on:

• Data Quality Engineering  
• Data Governance  
• Metadata Management  
• Data Lineage  
• Observability  

Dataset generated using Synthea synthetic healthcare data.

## Architecture

Source Systems → Raw → Staging → Curated → Governance → Analytics

## Technologies

- SQL
- Azure Data Lake
- Microsoft Purview
- dbt
- Power BI
- Python
- GitHub

Dataset size: ~2M+ healthcare records
